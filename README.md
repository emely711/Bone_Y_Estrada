##Descripción del proyecto

Este proyecto incluye el desarrollo de dos ejercicios aplicando Programación Orientada a Objetos en Java:

1. **Sistema de Subsidio de Gasolina**
   Permite evaluar si un ciudadano cumple con los requisitos establecidos por el Gobierno del Ecuador para acceder al subsidio, considerando ingresos, cantidad de vehículos y lugar de residencia.

2. **Sistema Bancario**
   Simula la gestión de diferentes tipos de cuentas bancarias (ahorros, corriente e inversión), aplicando herencia, encapsulamiento y polimorfismo para calcular intereses mensuales.

En ambos ejercicios se aplicaron conceptos como:

* Encapsulamiento
* Constructores
* Validación de datos
* Herencia
* Polimorfismo
* Sobrecarga de métodos

##Instrucciones para ejecutar el proyecto

1. Abrir el proyecto en IntelliJ IDEA o cualquier IDE de Java
2. Verificar la estructura de paquetes
3. Ejecutar las clases principales según el ejercicio:

   * `Main` para el sistema de subsidio
   * `MainBanco` para el sistema bancario
4. Ingresar los datos solicitados en consola
5. Observar los resultados generados
